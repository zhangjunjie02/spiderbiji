本周作业

本周作业

一、 问题描述：

Web前台界面：通过本周学习完成一套商城前台静态界面项目，要求内有商城首页，商品列表页，和商品信息详情页，共计三个页面。

二、 解题提示：

  • 由于我们不是专业美工，故本次作业可以仿现有网站界面来做。
  • 如：京东和淘宝商城（界面有点大）、小米官网，魅族手机网等。

三、 批改标准：

a) 首页：30分（导航、商品分类部分展示或最新、热卖、推荐等部分商品展示）。

b) 商品列表页：30分。

c) 商品详情页：30分。

d) 其他项：10分（文件结构清晰，代码整洁，要求适量的注释）。
